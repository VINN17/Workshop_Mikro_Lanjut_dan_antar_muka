2. Format Request dari PC

üì• Command dari PC ke Master

Format: XY (2 karakter ASCII)

X = Slave ID (1 digit)

  • 1 ‚Üí Slave 0x01
  • 2 ‚Üí Slave 0x02
  • 3 ‚Üí Slave 0x03

Y = Data Select (1 digit)

  • 1 ‚Üí PB2 only
  • 2 ‚Üí ADC only
  • 3 ‚Üí PB2 + ADC
  • 4 ‚Üí Counter only
  • 5 ‚Üí PB2 + Counter
  • 6 ‚Üí ADC + Counter
  • 7 ‚Üí ALL (semua data)

Contoh Request

Command Arti Output di Serial
12 Slave 1, ADC only >>> 523
27 Slave 2, ALL data >>> 1 487 12
35 Slave 3, PB2 + Counter >>> 0 8
STOP Hentikan display [Display stopped]

4. Slave: ISR State Machine

‚ö° 6 State Machine

Setiap byte SPI trigger interrupt ‚Üí ISR(SPI_STC_vect)

State 1: WAIT_SYNC1
Tunggu 0xFF ‚Üí jika terima ‚Üí State 2
State 2: WAIT_SYNC2
Tunggu 0xFE ‚Üí jika terima ‚Üí State 3, jika tidak ‚Üí State 1
State 3: WAIT_ID
Cek ID cocok ‚Üí jika ya ‚Üí State 4, jika tidak ‚Üí State 1
State 4: WAIT_CMD
Simpan command ‚Üí State 5
State 5: WAIT_CHECKSUM
Validasi checksum ‚Üí jika OK ‚Üí prepareResponse(), State 6
State 6: SEND_RESPONSE
Kirim byte per byte ‚Üí selesai ‚Üí State 1
// ISR dipanggil otomatis setiap SPI transfer ISR(SPI_STC_vect) { if(digitalRead(SS_PIN) == HIGH) { currentState = WAIT_SYNC1; // Reset jika SS HIGH return; } uint8_t received = SPDR; uint8_t toSend = 0x00; switch(currentState) { case WAIT_CHECKSUM: if(checksum == expectedChecksum) { prepareResponse(receivedCMD); currentState = SEND_RESPONSE; toSend = responseBuffer[0]; } break; } SPDR = toSend; // Load byte berikutnya }

6. Master Parsing Response

üîç Strategi: Cari SYNC Pattern

Master baca 15 bytes, scan untuk cari pattern 0xFF 0xFE [ID]

Kenapa 15 bytes? Antisipasi response tertunda
